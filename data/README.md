# Data

Run `python scripts/build_phase3_finance_data.py` to create the main public VIX dataset.

The final public repository should include:

- `processed/finance_vix_vol_h1.csv`
- `manifests/finance_vix_vol_h1.json`

The manifest records source, requested/actual dates, rows, split sizes, columns, and SHA-256 checksum. No proprietary data is used.
